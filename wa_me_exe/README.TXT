Para executar o programa, 
basta utilizar o executavel (wpp_me.exe) ,
que se encontra no arquivo wpp_me.rar nesta pasta,
para melhor comodidade, crie um atalho do executável na sua área de trabalho, ou onde preferir

Este aplicativo funciona pela API oficial do Whatsapp, 
apenas acessando seu endereço na internet , 
abrindo o navegador padrão de forma automática para o usuário, 
após ele digitar o número desejado com um DDD válido e o telefone com a quantidade correta de caracteres
(Validação feita somente para números do Brasil)
